# 📄 PDF Rename Tool (Preview付き)

PDFを開かずに中身を確認しながらリネームできるツールです。  
AI（ChatGPT）を使いながら作成しました。

---

## 🚀 特徴

- 📄 PDFの1ページ目をプレビュー表示
- 📁 フォルダ内のPDFを一覧表示
- ✏️ 開かずにそのままリネーム
- ⚡ シンプルで使いやすいGUI（Tkinter）

---

## 🖥 デモ

※ここにスクリーンショットを貼ると分かりやすいです

---

## 📦 必要環境

- Python 3.x

---

## 📥 インストール

pip install -r requirements.txt

---

▶️ 使い方

rename.py
1.フォルダを選択
2.PDFをクリックしてプレビュー表示
3.ファイル名を入力してリネーム
